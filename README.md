## Як запустити

### Problem 1 (масиви / мутації)

1. Перейдіть у папку:
   ```bash
   cd Problem_1
   ```
2. Встановіть залежності (для jQuery):
   ```bash
   npm install
   ```
3. Запустіть будь‑який простий статичний сервер і відкрийте сторінку у браузері (або просто відкрийте `index.html` як файл):
   ```bash
   npx http-server -c-1 -p 5174 .
   # відкрийте http://localhost:5174
   ```

### Problem 2 (iOS калькулятор на React)

1. Перейдіть у папку:
   ```bash
   cd problem_2
   ```
2. Запустіть локальний сервер (через вбудований скрипт):
   ```bash
   npm run start
   # або: npm run serve
   ```
3. Відкрийте у браузері:
   ```
   http://localhost:5173
   ```

Примітка: Problem 2 використовує React UMD + Babel у браузері, тож збірка не потрібна — достатньо відкрити `index.html` через статичний сервер.
